## ðŸŽ¨ Essential Features

### Tabs
- **Open new tab**: `Ctrl+T`
- **Close tab**: `Ctrl+W`
- **Switch tabs**: `Ctrl+Tab` / `Ctrl+Shift+Tab`
- **Close all tabs**: `Ctrl+Shift+W` (with confirmation)
- **Drag to reorder**: Click and drag tab headers

### Search
- **Find in document**: Press `Ctrl+F`
- **Next/Previous match**: `F3` / `Shift+F3`
- **Type to filter files**: Use the search box in the file tree sidebar

### Navigation
- **Go to heading**: `Ctrl+G` - Quick jump to any heading in the document
- **Back/Forward**: `Alt+Left` / `Alt+Right`
- **Refresh**: `F5`

### Themes
- **Auto-detect**: Follows your Windows theme by default
- **Manual toggle**: Use Settings to switch between Light, Dark, and High Contrast modes

### Accessibility
- **Screen reader support**: Full ARIA annotations
- **Keyboard navigation**: Every feature is accessible via keyboard
- **High contrast**: Optimized colors for accessibility
- **Tooltips**: Hover over buttons for helpful hints

## ðŸŽ¯ Pro Tips

### Sidebar Management
- **Toggle sidebar**: Click the sidebar button or use the View menu
- **Resize sidebar**: Drag the splitter between sidebar and content
- **File tree search**: Start typing to filter files instantly

### Link Navigation
- **Internal links**: Click markdown links to navigate between documents
- **External links**: URLs open in your default browser (with confirmation)
- **Copy links**: Right-click and copy, or use `Ctrl+Shift+C` for file paths

### Content Viewing
- **Zoom**: Use your browser-style zoom controls (coming soon)
- **Scroll sync**: Your scroll position is remembered per document
- **Code highlighting**: 180+ languages supported automatically
- **Mermaid diagrams**: Rendered inline with syntax checking

### Session Recovery
- **Auto-save**: Your open tabs and scroll positions are saved automatically
- **Crash recovery**: MarkRead will offer to restore your session after unexpected closures
